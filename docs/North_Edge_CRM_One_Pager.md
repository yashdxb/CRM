## What Is It?

North Edge CRM is a **modern, all-in-one CRM platform** that combines sales, marketing, customer support, and AI intelligence into a single system — built for mid-market and enterprise businesses.

---

## Key Capabilities

| Area | What You Get |
|------|-------------|
| **Sales & Pipeline** | Lead management, opportunity tracking, quote generation, deal approvals, and automated renewals |
| **AI Intelligence** | AI-powered lead scoring, smart assistant with action execution, campaign recommendations |
| **Marketing** | Campaign management, attribution tracking, health scoring, email templates |
| **Help Desk** | Support ticketing, queue management, SLA enforcement, automatic escalation |
| **Automation** | Visual drag-and-drop workflow builder, approval chains, background task processing |
| **Reports** | Built-in report designer, customizable dashboards with 18 widget types |
| **Email** | Full Microsoft 365 & Gmail integration (send, receive, sync) |
| **Team Chat** | Built-in real-time messaging — no need for separate tools |
| **Security** | Role-based access (29 permissions), multi-tenant isolation, full audit trail |
