# North Edge CRM Mobile

Separate React Native mobile app for the CRM, scaffolded with Expo and TypeScript.

## Run

```bash
cd mobile
npm start
```

Then launch one of:

- `npm run ios`
- `npm run android`
- `npm run web`

## Purpose

This app is intended to evolve into the dedicated mobile workspace for:

- Leads
- Contacts
- Deals
- Activities
- Approvals

## Notes

- This project is intentionally separate from the Angular web client in `client/`.
- Shared types, API clients, and utilities can be extracted later into shared packages if needed.
